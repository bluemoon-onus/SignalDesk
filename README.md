# SignalDesk

SignalDesk is a business-first demo for enterprise AI sales.

It is built with Next.js 14 App Router, TypeScript, Tailwind CSS, and shadcn/ui, and frames each screen as an executive decision surface instead of a chatbot workflow.

## Routes

- `/snapshot`
- `/opportunities`
- `/strategy`
- `/pilot`

## Local commands

```bash
npm install
npm run lint
npm run typecheck
npm run build
npm run start -- --hostname 127.0.0.1 --port 3000
```
